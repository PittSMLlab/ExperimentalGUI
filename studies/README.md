# studies/

Protocol scripts for each experimental study. Each subfolder contains
speed-profile generators, protocol runner scripts, and data-transfer
utilities. Refer to the study's own README for experimenters, current
status, and key scripts.

For the controller reference, hardware setup, and protocol creation
guide, see [EXPERIMENT_SETUP.md](../EXPERIMENT_SETUP.md).
