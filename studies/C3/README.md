# C3 — Context Specificity, Cognition, and Connectivity

Examines split-belt treadmill locomotor adaptation in participants
with chronic stroke alongside age-matched controls. C3 is Nate
Brantly's doctoral thesis project.

## Status

**Active.** Approximately 3 control participants remain (6 visits
total; each control completes two visits). Estimated completion:
early July 2026.

Consult the PI before making functional changes to any script in
this folder.
